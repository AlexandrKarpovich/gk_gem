Сделать копию вёрстки шапки с сайта http://www.winnis.it/en/;

1. Сверстать шапку. 
1.1. В верхнем элементе убрать соцсети и языки.
    Оставить поиск. Переделать интерфейс поиска. При нажатии на search, появится поле input с placeholder="search". При клике вне поля, input скрывается. Дизайн на усмотрение, в рамках цвета сайта.
1.2. Тело шапки, сделать как есть на сайте (http://www.winnis.it/en/), липкую, с анимацией сжатия при прокрутке.
1.3. Сделать выпадающее меню на 1 из пунктов главного меню.


2. Адаптировать под устройства с шириной меньше 768px;
2.1. Мобильная шапка оставить как есть на сайте  (http://www.winnis.it/en/), сделать анимацию бутерброда при нажатии.
2.2. В меню мобильной шапки, оставить только пункты главного меню, без выпадения. Само меню должно открываться без смещения <main>


P.S.
1.Используй любые библиотеки.
2.Сделать темный фон для контраста с шапкой.
3.Необязательно! Сделать полноэкранный слайдер с адаптивом. Для устройств с шириной меньше 768px. Показывать или отдавать необходимое изображение.
4.Выполненое задание прислать в виде архива.
